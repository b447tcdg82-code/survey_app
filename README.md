# Опрос: Отношение к геймификации в образовании

Одностраничный веб-опросник, созданный в рамках учебной практики.  
Стек: Python, Streamlit, Firebase Firestore, Pandas, Plotly.

## Тема

Исследование отношения студентов к геймификации в образовании:
вовлечённость, мотивация, справедливость системы баллов, влияние рейтингов на стресс.

## Запуск локально

1. Клонируй репозиторий:
```bash
git clone https://github.com/ВАШ_НИКНЕЙМ/survey_app.git
cd survey_app
```

2. Установи зависимости:
```bash
pip install -r requirements.txt
```

3. Добавь файл `serviceAccountKey.json` в корень проекта (получить в Firebase Console → Project Settings → Service Accounts).

4. Запусти приложение:
```bash
streamlit run app.py
```

5. Открой в браузере: http://localhost:8501

## Структура проекта

```
survey_app/
```

## Вопросы опроса

- Возраст и пол
- Уровень образования
- Опыт с геймификацией в обучении
- Вовлечённость при использовании геймификации (1–10)
- Любимые элементы геймификации (баллы, ачивки, рейтинги и др.)
- Влияние на мотивацию
- Справедливость системы баллов (1–10)
- Влияние рейтинга на уровень стресса
